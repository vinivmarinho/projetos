Variáveis:
- display => guarda o espaço onde os itens serão mostrados
- resultado => guarda o espaço onde o resultado da operação será mostrado
- numeros => Se refere a todos os botões que são números
- operadores => Se refere a todos os botões que são operadores
- ponto => Ponto (.)
- primeiro_operando => Dígitos usados antes do operador ou o resultado de uma operação
- Digitos => Guarda o(s) número(s) que o usuário está escolhendo
- Igual => Sinal(=)
- resultado_operacao => Resultado das operações

Fluxo de Execução:
1-> Usuário escolhe os números(12)
- Função: Adiciona_numero(12)
- "Digitos" = 12
- "resultado" mostra digitos

2-> Usuário escolhe um operador (+ - x ÷)
- Função: adiciona_operador
- Se o "primeiro_operando" e "digitos" não estiverem vazios:
    - "resultado_operacao" recebe o retorno da função "operacao"
    - "resultado" mostra "resultado_operacao"
    - "Primeiro_operando" = "resultado_operacao"
    - Limpa "digitos"
